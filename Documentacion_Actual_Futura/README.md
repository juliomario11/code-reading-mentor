# Documentación actual y futura

Esta carpeta contiene material complementario al README principal: investigación reciente,
ideas para extender el starter y preguntas para explorar más a fondo. **No es necesario leerla
para correr el proyecto** — el README ya cubre el camino feliz. Esto es para cuando quieras
profundizar.

## Sugerencia de orden de lectura

Si eres un dev junior que recién corrió el starter y quieres seguir aprendiendo:

1. **Empieza por** [`mas_ejemplos.md`](./mas_ejemplos.md) — para ver hasta dónde se puede
   estirar la misma plantilla cambiando solo el system prompt.
2. **Después** [`agentes_vs_mcp.md`](./agentes_vs_mcp.md) — porque MCP es el siguiente paso
   natural cuando tu agente necesita herramientas (no solo texto).
3. **Cuando te preguntes** "¿y si el modelo base no me alcanza?", lee [`fine_tuning.md`](./fine_tuning.md).
4. **Para entender el contexto general**, ve a [`tendencias_2026.md`](./tendencias_2026.md).
5. **Para retarte a ti mismo**, abre [`preguntas_para_explorar.md`](./preguntas_para_explorar.md).
